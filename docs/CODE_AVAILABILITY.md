# Code Availability

The source code, configuration files, environment specification, exported manifests, and analysis scripts are publicly available at https://github.com/Lekzyboi/Class-Conditional-CP.

The version used for this manuscript has been archived at Zenodo with DOI https://doi.org/10.5281/zenodo.21362694.

The framework is organized to reproduce the conformal analyses from archived probability arrays and labels without retraining the base classifier.
